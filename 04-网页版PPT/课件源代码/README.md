# 课件源代码

这里是网页版课件唯一的工作目录。不要为 v1、v2 复制多套源代码。

- 日常迭代：直接修改当前源代码并提交 Git。
- 阶段冻结：创建 `ppt-YYYY-MM-DD-vN` tag。
- 交付导出：把 PPTX、PDF 或离线 HTML 包写入 `06-交付成果/`，并登记 tag 和 commit。
- 讲稿：维护在 `05-讲稿/讲稿.md`，与课件源代码在同一个 tag 下自然对应。

## 当前内容

- `goal.json`：当前 12 页课件配置，页面物理顺序对应 `A-01—A-12`。
- `index.html`：可编辑、可导出的网页课件；运行时页面 ID 尾号与实际页码一致。
- `assets/`：课件运行资源、品牌标识和页面图示。

## 页码与技术标识

- `A-xx` 等于实际物理页码，章节封面也占用连续编号。
- `theme07_pageNNN` 是稳定布局键，不代表页码，不随页面移动而重命名。
- 新增、删除或移动页面后，必须同步更新 `goal.json`、运行时 `slideOrder`、讲稿、大纲、页面台账和相关素材引用。
