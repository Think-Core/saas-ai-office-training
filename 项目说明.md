# SaaS 类 AI 工具赋能基础办公

## 项目目标

为公司业务骨干 AI 实训第一期课程 1 制作一套约 60 分钟的中文培训课件、逐页讲稿、录屏案例和配套资料。

授课人：张金京，人民邮电出版社 AIGC 项目组组长，负责 AIGC 技术跟踪、前沿技术研究、体系规划和产品规划设计。

## 课程定位

课程不以罗列工具或教授提示词为目标，而是帮助非技术业务骨干理解 AI 从“回答问题”到“完成任务”的变化，并掌握可以迁移到自身工作的 Agent 办公方法。

主要演示工具为 WorkBuddy。正式案例采用预录屏形式，播放前先用图文讲清任务逻辑；现场只预留一个可跳过的加演与互动模块。

## 交付物

- 可在浏览器打开和迭代的 16:9 HTML 课件
- 与 HTML 源代码在同一 Git 版本下对应的 Markdown 讲稿
- WorkBuddy 安装、配置、界面和案例录屏及关键帧截图
- 可编辑 PPTX 与 PDF 阶段成果
- 可追溯的资料索引、决策记录、页面台账和跨 task 交接记录

## 当前视觉方向

- Dashi PPT `theme07` 冷白调研风
- 科技蓝为主，少量红色用于风险和关键提示
- 页面强调清晰、克制、易讲解，不把演示文稿做成操作手册式的密集截图集合

## 版本管理

- 网页源代码和讲稿：单一工作目录，由 Git commit 管理。
- 阶段冻结：`ppt-YYYY-MM-DD-vN` Git tag。
- PPTX、PDF、离线 HTML 包：`YYYY-MM-DD-vN-SaaS类AI工具赋能基础办公`。
